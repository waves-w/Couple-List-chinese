Lottie 动画资源
================
将你的 Lottie JSON 文件放在此目录即可在「链接加载页」显示为动画。

• 默认动画文件名为：link_loading.json（已包含一个简单加载动画）
• 替换为自己的动画：用你的 .json 覆盖 link_loading.json，或修改 LinkLoadingView 中的 kLottieAnimationName
• 可从 https://lottiefiles.com 下载免费 Lottie 动画（导出为 JSON）

此目录已加入工程，编译时会自动复制到 App Bundle 的 Animations 子目录。
